# 核心防御者 (CORE DEFENDER) - 游戏策划文档
**版本:** 2.2 (遗物更新版)

## 1. 游戏概述
**核心防御者** 是一款快节奏的街机风格塔防/Roguelite生存游戏。玩家控制一个位于屏幕中心的固定核心，通过自动射击（支持手动辅助瞄准）来抵御源源不断的几何体敌人。

**核心体验:** "爽快割草" + "策略构建" + "永久成长"。

---

## 2. 核心循环 (Core Loop)
1.  **防御 (Defend):** 敌人从屏幕边缘生成并向中心移动。如果接触核心，核心扣除 HP。HP 归零游戏结束。
2.  **击杀与收集 (Kill & Collect):** 击杀敌人掉落经验球 (XP Orbs) 和 宝箱 (Chests)。
3.  **升级 (Upgrade):** 收集经验球填满经验条。升级时从 3 个随机选项中选择一个（属性提升或技能Tag）。
4.  **遗物 (Relics):** 拾取宝箱获得强力被动道具 "遗物"。Boss 必掉，小怪概率掉落。
5.  **协同 (Synergize):** 收集特定的 Tag 组合会自动激活强大的被动 "羁绊 (Synergy)"。
6.  **局外成长 (Meta-Progress):** 游戏结束后根据分数获得 "科技点 (TP)"，用于在主菜单购买永久属性加成。

---

## 3. 游戏机制

### 3.1 玩家 (核心)
*   **生命值 (HP):** 核心的生存指标。
*   **攻击:** 自动射击最近的敌人，或跟随鼠标/触摸位置射击。
*   **过载 (Overload/Ult):** 击杀敌人积累能量。能量满时手动释放，进入 5 秒 "无敌爆发模式"。
*   **护盾 (Shields):** 独立旋转的轨道防御系统，不受核心瞄准方向影响。

### 3.2 敌人设计
(同前个版本)
*   **AI更新:** 敌人现在拥有智能寻路，遇到障碍物时会尝试沿墙滑动，而不是被卡住。

### 3.3 物品与掉落
*   **经验球 (XP Orb):** 绿色光球，提供经验。
*   **宝箱 (Chest):** 金色立方体。拾取后暂停游戏，展示获得的随机遗物。
    *   Boss: 100% 掉落。
    *   精英/普通怪: 极低概率掉落。

### 3.4 升级系统 (Roguelite)
(同前个版本)

### 3.5 遗物系统 (New)
遗物提供独特的、通常无法通过升级获得的被动效果：
*   **吸血鬼之牙:** 击杀回血。
*   **反应堆:** 增加射速。
*   **瞄准镜:** 增加暴击率。
*   **磁铁:** 增加拾取范围。
*   **装甲板:** 增加防御/血量。

## 4. 控制与调试
*   **移动/瞄准:** 鼠标或触摸拖动。
*   **暂停:** ESC 键或点击右上角暂停按钮。
*   **调试信息:** F3 键切换 FPS 和统计数据显示。

## 5. 视觉与反馈
*   **飘字:** 伤害数字现在会从敌人头顶飘出。白色为普通，黄色为暴击。
*   **UI:** 右侧栏实时显示已获得的技能标签(Tags)和遗物(Relics)。
